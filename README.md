### 빵집 가상 시뮬레이션 게임

&nbsp;&nbsp;이 프로젝트는 실제 빵집 종업원의 환경에서 일어날 수 있는 환경을 구현해보고자 제작한 프로젝트입니다.

<br/>

---

#### 개요

##### **가게 OPEN 전**

- OPEN전 아래와 같은 기본적인 업무를 진행합니다.
  - 빵 연구소(신제품 등록, 변경, 취소, 조회)
  - 재고조회
  - 제품 검색
  - 폐기 처리
  - 매출 장부 조회

##### **가게 OPEN 후**

- 가게 OPEN시 손님이 랜덤한 시간 간격으로 손님이 입장하며 업무는 아래와 같습니다.
  - 주문 조회
  - 빵 제작
  - 빵 판매
  - 손님 마감
